# Phone_shop
An E-commerce website for phone parts using ASP.NET MVC.

[Live Demo](http://5mshop.somee.com/)

## Features

- Authentication & Authorization:
- Profile
- Searching & Filtering
- Notifications
- Cart
- Order history
- Admin Section
- Delivery Section

## Frontend technologies:
- HTML
- CSS
- JavaScript

## Backend technologies:
- C#
- ASP.NET
- Sql Server
